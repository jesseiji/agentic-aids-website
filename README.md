# Agentic Aids Website

Custom-designed website for **Agentic Aids** — a software engineering business that designs, builds, and maintains custom websites for small businesses.

## Pages

- **Home** — Hero with typing animation, services overview, process steps
- **Tools** — 9 service offerings (design, SEO, ordering, maintenance, etc.)
- **Portfolio** — 6 project showcases with tech stacks
- **Pricing** — 3 tiers: Starter ($999), Growth ($2,499), Premium ($4,999)
- **Contact** — Contact form + business info
- **Get Started** — Client intake form + process timeline
- **Login** — Login/signup with client-side auth (localStorage)

## Tech Stack

- HTML, CSS, vanilla JavaScript
- Google Fonts: Space Grotesk, Inter, JetBrains Mono
- Dark mode theme with coding-inspired aesthetic
- Fully responsive (mobile, tablet, desktop)

## Setup

Open `index.html` in a browser — no build step required.
